# Recruit ERP 변경 이력

## v10.40.29 SCHOOL_EXACT_AUTO_LINK
- 학교 연결 필요 영역에 `확실한 항목 자동 연결` 기능 추가
- 학교명 정확 일치 또는 하나의 학교에만 일치하는 별칭만 자동 연결 후보로 분류
- 같은 학교명이 여러 ID에 존재하거나 별칭이 여러 학교에 중복된 경우 자동 연결 제외
- 띄어쓰기·특수문자 차이 및 유사도 후보 자동 연결 금지
- 지원자·사원별 연결 대상 미리보기와 선택 적용 제공
- 적용 전 전체 ERP JSON 안전백업 다운로드
- 기존 유효 schoolId는 덮어쓰지 않음
- 학교 ID와 학교 마스터 구조 변경 없음
- 학교 연결 필요 집계가 정확 일치 미연결 데이터도 포함하도록 보완
