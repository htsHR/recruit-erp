# v10.38.5 UI_CLEANUP

- 새 기능 추가 없이 v10.38.4 기능을 유지한 채 화면 밀도를 정리했습니다.
- 사이드바에서 고급검색·중복관리·안내문 템플릿의 독립 메뉴 노출을 제거했습니다.
- 고급검색은 지원자 관리의 `상세필터 열기` 버튼으로 진입합니다.
- 데이터 점검과 중복 후보를 `데이터 관리` 내부 탭처럼 이동하도록 정리했습니다.
- 홈 업무 카드 수, 영어 장식 문구, 카드 그림자와 테두리를 축소했습니다.
- 알림·도움말 등 동작하지 않는 상단 아이콘을 숨겼습니다.
- localStorage 키, JSON 구조, Supabase 저장 로직은 변경하지 않았습니다.
