# Recruit ERP v10.40.1 HOME_DEV · EXCEL_ROW_PASTE

## 기준
- v10.40.0 MODULE_SPLIT 통교체본 기준
- 기존 localStorage 키, JSON 구조, Supabase 저장 구조 유지

## 추가 기능
- 신규 지원자 등록 화면에 `엑셀 행 붙여넣기` 추가
- 구직 지원자 명단 A~V 22열 한 행을 탭 구분값으로 자동 분석
- 엑셀 헤더 + 지원자 한 행을 함께 복사해도 자동으로 헤더 제외
- 셀 내부 줄바꿈과 큰따옴표가 포함된 경력·비고도 안전하게 분석
- 지원일, 면접일, 입사일, 면접시간 자동 변환
- 연락상태, 성별, 경력구분, 학력구분, 출근방법 자동 보정
- 지역(시) + 세부지역(동)을 ERP 지역 한 칸으로 결합
- 지원파트를 기존 `extra` 필드에 저장할 수 있도록 등록 폼에 표시

## 검토 및 수정
- 붙여넣은 값을 별도 편집 화면에서 직접 수정 가능
- 형식이 불확실하거나 직접 선택이 필요한 값은 노란색으로 표시
- 연락처·이메일·성명+생년 기준 중복 후보 사전 표시
- 붙여넣기만으로 자동 저장하거나 기존 데이터를 자동 덮어쓰지 않음

## 기존 지원자 수정
- 지원자 수정 화면에서 엑셀 행 붙여넣기 가능
- 현재 값과 붙여넣은 값을 항목별로 비교
- 값이 다르고 엑셀에 실제 값이 있는 항목만 기본 선택
- 사용자가 선택한 항목만 수정 폼에 적용한 뒤 기존 `수정 저장`으로 확정

- 기존 JSON 백업·복원 호환 유지
- Supabase 및 회사/집 운영 모드 로직 변경 없음
