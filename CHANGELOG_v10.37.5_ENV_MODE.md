# Recruit ERP v10.37.5 HOME_DEV · ENV_MODE

## 수정 배경

`회사 로컬 운영 모드` 안내가 고정 문구여서 집에서도 회사 운영 안내가 표시되는 문제를 수정했습니다.

## 변경 사항

- 사이드바에 회사/집 환경 선택 버튼 추가
- 회사 선택 시 `회사 로컬 운영 모드` 및 `COMPANY MASTER` 표시
- 집 선택 시 `집 개발·복원 모드` 및 `HOME DEV` 표시
- 환경 선택값을 브라우저별 UI 설정 localStorage에 저장
- HOME_DEV 배포본의 최초 기본값을 집 모드로 지정

## 데이터 영향

- 지원자·협력학교·사원명부·일정 데이터 구조 변경 없음
- 기존 핵심 localStorage 저장키 변경 없음
- Supabase 및 JSON 백업·복원 로직 변경 없음
