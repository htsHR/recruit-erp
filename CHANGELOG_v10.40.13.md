# Recruit ERP v10.40.13 HOME_DEV · EMPLOYEE_UI_DETAIL_FINAL

## 변경 사항
- 사원명부 상단 소개·요약 카드·검색 영역을 컴팩트하게 재구성했습니다.
- 전체/재직/휴직/퇴직/입사예정 카드 클릭 필터를 추가했습니다.
- 이름·사번·팀/부서·직급·재직상태·출신학교·입사일 기간 필터를 구성했습니다.
- 목록 열을 사번/성명/소속/직급/입사일/근속기간/출신학교/재직상태/관리로 정리했습니다.
- 목록의 삭제 기능을 제거하고 수정 화면의 위험 영역으로 이동했습니다.
- 사원 상세·등록·수정 화면에 기본/조직/채용/학력/기타 선택 필드를 추가했습니다.
- 지원자 기록과 협력학교 연결 상태를 상세 화면에서 확인할 수 있습니다.
- 기존 사원 데이터와 JSON은 새 필드가 없어도 정상적으로 표시됩니다.
- Supabase 확장필드 마이그레이션 전에는 기존 필드 저장으로 자동 대체됩니다.

## 데이터 안전
- 기존 EMPLOYEES_KEY와 기존 필드를 유지합니다.
- 주민등록번호·주소·개인 연락처·개인 이메일은 저장하거나 표시하지 않습니다.
- 지원자 목록과 공용 목록 레이아웃은 변경하지 않았습니다.
