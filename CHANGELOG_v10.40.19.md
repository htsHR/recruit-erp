## 변경 사항

- 사원명부 상단 소개·업무 버튼 정돈
- 전체/재직/휴직/퇴직/입사예정 요약 카드를 상태 필터로 유지하면서 크기와 정보 위계 재설계
- 상태 데이터 점검 영역을 한 줄형 알림으로 축소
- 목록/조직별 현황 전환과 협력학교 연계 버튼 정돈
- 검색 영역을 흰색 카드로 변경하여 진한 파란 작업바 제거
- 기본 검색을 이름/사번/팀/재직상태로 단순화
- 그룹/제품/파트/직급/학교/입사일/저장조건을 상세 필터 안으로 이동
- 검색 결과와 조직 종류 수를 작은 요약 칩으로 정리
- 중복된 목록 내 사원 추가/내보내기 버튼은 화면에서 숨기고 상단 도구 모음으로 통합
- 사원 목록 9개 열의 폭·글자 크기·행 높이·상태 배지·관리 버튼 정렬 개선
- 데스크톱 표 셀을 flex로 변경하던 규칙을 제거하여 행 높이와 관리 버튼 위치 안정화
- 1450px 이하 화면에서 근속기간/출신학교 열을 숨겨 관리 열 우선 보존
- 모바일 사원 카드를 전체 폭으로 고정하고 상세/수정 버튼을 같은 줄에 배치
