## 추가

- 목록 행 키보드 접근 기능
  - `Tab`으로 행 선택
  - `Enter` 또는 `Space`로 상세보기
- 모바일 목록용 `더보기` 메뉴
- 사원 상태 탭별 인원수 배지
- 사원 검색 결과 요약 표시
- 사원 검색창 Enter 실행
- 목록 외부 클릭 시 열린 `더보기` 메뉴 닫기
