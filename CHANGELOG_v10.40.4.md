# Recruit ERP v10.40.4 · EXCEL_UPDATE_COMPARE

- 기존 지원자 수정 시 현재 값과 엑셀 값을 비교하고, 달라진 항목만 표시합니다.
- 변경할 항목만 체크해 입력폼에 반영할 수 있습니다.
- 엑셀의 빈 셀은 기존 지원자 값을 지우지 않습니다.
- 메모는 기존 메모 뒤에 추가하거나 엑셀 메모로 교체할 수 있습니다.
- 연락처가 정확히 일치하는 기존 지원자가 1명일 때 자동으로 수정 대상에 연결합니다.
- 적용 전 `현재 값 → 변경 값` 목록을 확인할 수 있습니다.
- 지원자 입력 화면의 `지원파트`, `기수/배치` 입력칸을 제거했습니다.
- 엑셀의 `지원파트` 열은 ERP의 `지원근무지`로 매핑합니다. 천안·평택·기타 외 값은 확인 전 적용할 수 없습니다.
- 기수/배치 통계와 CSV의 불필요한 열을 제거했습니다.
- 기존 JSON의 `extra`, `batch` 값은 하위 호환을 위해 보존하지만 신규 입력·수정 UI에서는 사용하지 않습니다.
