# Recruit ERP v10.36.4 HOME_DEV — Unified Corporate UI

## 변경 범위
- 좌측 메뉴바와 상단 헤더를 전 페이지 공통 기업형 레이아웃으로 통일
- 전체 기본 글자 크기, 입력창, 버튼, 카드, 테이블 밀도를 상향 조정
- 협력학교 관리 KPI, 네이비 필터 패널, 학교 목록 테이블을 시안에 가깝게 재구성
- 홈, 지원자, 오늘 할 일, 일정, 통계, 사원명부, 템플릿, 백업 페이지의 공통 카드·패널 톤 통일
- 페이지별 상단 설명 문구 동적 적용

## 데이터/기능 영향
- LocalStorage 키 변경 없음
- Supabase 동기화 로직 변경 없음
- 학교·직원·지원자 데이터 구조 변경 없음
- 학교 ID 및 직원 schoolId 연결 로직 변경 없음
