# CHANGELOG v10.37.4 HOME_DEV · COMPLETE_UX

기준: `v10.36.6 HOME_DEV · LIST_UX`

앞서 계획한 v10.36.7~v10.37.4 UI/UX 항목을 하나의 통합 릴리스로 적용했습니다.

## 1. WORKFLOW_UX

- 홈 업무 카드를 오늘 면접·기한 경과·연락 필요·판정 필요·입사 임박으로 정리
- 홈 KPI와 업무 카드를 관련 화면으로 연결
- 오늘 할 일에서 빈 업무 그룹 자동 숨김
- 기한 경과 전용 긴급 패널 추가
- 업무 카드에서 지원자 상태를 바로 변경
- 면접예정·입사예정 변경 시 관련 날짜 입력 지원
- 종료 상태 변경 시 사유 메모 입력 지원

## 2. FORM_UX

- 신규/수정 모드 배너 추가
- 입력 진행률과 섹션별 작성 진행 표시
- 연락처 자동 하이픈
- 성명·전화번호·이메일 인라인 검증
- 미저장 상태에서 페이지 이동 및 브라우저 종료 경고
- 초기화 전 확인
- 등록·수정 완료 토스트

## 3. DETAIL_VIEW

- 지원자 상세에 진행 단계 표시
- 상세 화면에서 빠른 상태변경
- 해당 지원자를 선택한 안내문 화면으로 바로 이동
- 상세 프로필 인쇄 버튼

## 4. CALENDAR_UX

- 월간·주간·목록 보기 추가
- 근무지 필터와 별도로 일정 종류 필터 추가
- 면접·입사·직접 일정·매우중요 필터 지원
- 모바일에서는 목록 보기를 기본 표시
- 주간/목록 일정에서 지원자 상세 또는 직접 일정 수정으로 이동

## 5. STATISTICS_UX

- 이번 달 지원·면접·합격·출근·합격률·평균 진행일 KPI 추가
- 핵심 요약, 입사 추이, 전환 및 채널 성과 섹션으로 재구성
- 근무지 필터를 상단 고정형으로 정돈

## 6. TEMPLATE_UX

- 지원자 선택 기능 추가
- 이름·근무지·면접/입사 일정 자동 입력
- 담당자·날짜·시간 직접 조정
- 문자 형태 미리보기
- 글자 수 표시
- 최근 생성 문구 8개 보관 및 다시 불러오기
- 상세 화면에서 선택 지원자와 함께 안내문 작성 화면 진입

## 7. SCHOOL_EMPLOYEE_UX

- 협력학교 관리 필요 기준 안내
- 담당자/연락처/상태/관리이력/연락일/학교구분 문제 태그 표시
- 중요 학교 별표 표시 기능
- 사원 등록·JSON 가져오기·내보내기 버튼 분리
- 사원 상세에서 재직상태 변경
- 퇴사 처리 시 퇴사일 입력

## 8. ACCESSIBILITY_POLISH

- 포커스 표시 강화
- 모바일 버튼 터치 영역 확대
- 공통 토스트 알림
- 상세 화면 인쇄 스타일
- 카드·패널·여백·텍스트 위계 정돈
- 모바일 일정·템플릿·업무 카드 반응형 보완
- 동작하지 않는 상단 알림/도움말 아이콘 숨김

## 로컬 운영 UI

- 사이드바에 `회사 로컬 운영 모드` 안내 적용
- 상단에 `LOCAL MASTER` 표시
- 클라우드 전용 로그인·스냅샷 화면은 운영 UI에서 숨김
- Supabase 코드와 핵심 저장 함수 자체는 변경하지 않음

## 데이터 구조

변경 없음:

- 지원자 구조 및 저장키
- 학교 구조 및 저장키
- 사원명부 구조 및 저장키
- 일정 구조 및 저장키
- 기존 JSON 백업·가져오기 형식

UI 전용 별도 키만 추가:

- `recruit_erp_ui_template_history`
- `recruit_erp_ui_school_favorites`
