# Recruit ERP v10.43.0 HOME_DEV

## SCHOOL_ANALYTICS_PACK

기준본: v10.42.0 HOME_DEV SCHOOL_RELATIONSHIP_PACK

### 추가 기능

- 협력학교 화면에 `성과 분석` 탭 추가
- 학교별 지원자·면접자·입사자·재직자·퇴직자 집계
- 지원 대비 입사 전환율 계산
- 사원명부 입사일·퇴사일을 이용한 평균 근속기간 계산
- 최근 1년 입사 실적 집계
- 최근 5개년 지원·면접·입사 비교
- 선택 연도 기준 지원근무지별 성과
- 선택 연도 기준 학과별 성과 상위 10개
- 학교 유형·연도·지원근무지·학교명 검색 필터
- 학교별 성과 CSV 내보내기
- 핵심학교 참고점수 및 표본 주의 표시

### 핵심학교 참고점수

- 지원 규모 20점
- 입사 실적 25점
- 현재 재직 20점
- 입사 전환율 15점
- 평균 근속 10점
- 최근 1년 입사 10점

참고점수는 화면 표시와 정렬에만 사용하며 학교 상태를 자동 변경하지 않습니다.
지원·입사·재직 표본이 모두 3명 미만인 학교는 `표본 주의`로 표시합니다.

### 데이터 보호

- 학교 ID 변경 없음
- 지원자 `schoolId` 변경 없음
- 사원 `schoolId` 변경 없음
- 기존 학교·지원자·사원 데이터 자동 수정 없음
- 신규 Supabase 컬럼 없음
