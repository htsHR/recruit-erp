# Recruit ERP v10.36.5 HOME_DEV — UI_LAYOUT

## 작업 범위
- UI/UX 및 반응형 레이아웃만 수정
- localStorage 데이터 구조, 저장 키, Supabase 로직, JSON 형식은 변경하지 않음

## 주요 변경
- 홈 화면 KPI를 핵심 4개 항목으로 정리
- 업무 카드 중복을 줄이고 정보 우선순위 재배치
- 공통 여백, 카드, 버튼, 글자 굵기와 높이 통일
- 지원자 목록 검색·필터 영역 정돈
- 지원자 테이블 가독성, 행 구분, 상태·관리 버튼 개선
- 1450px 이하에서 저우선 열 자동 축소
- 760px 이하에서 지원자 목록을 카드형 레이아웃으로 전환
- 신규 지원자 등록 섹션 헤더와 입력 폼 간격 정돈
- 모바일 상단 버튼, KPI, 업무 카드 반응형 보완
- 실제 기능이 없는 알림·도움말 아이콘 숨김
