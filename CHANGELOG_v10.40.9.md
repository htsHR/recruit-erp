# Recruit ERP v10.40.9 HOME_DEV · DATA_HEALTH_FINAL

## 데이터 점검센터 최종화
- 지원자 성명·연락처 누락 및 연락처·이메일 형식 검사
- 생년/생년월일 형식 검사
- 지원근무지·출근방법 미확인 검사
- 상태와 면접일·입사일 불일치 및 날짜 순서 검사
- 오래된 미처리 지원자 검사
- 지원자·사원의 학교명과 schoolId 연결 불일치 검사
- 출근 처리 지원자의 사원명부 누락 검사
- 퇴직자 퇴사일 누락 및 사원 상태와 입·퇴사일 불일치 검사
- 오류 / 주의 / 확인 필요 분리
- 지원자 / 학교 연결 / 사원 영역 필터
- 문제별 상세·수정 화면 이동
- 자동 수정 없음

## 엑셀 붙여넣기 추적
- v10.40.9 이후 엑셀로 신규 등록하거나 수정한 지원자 ID를 별도 localStorage 키로 추적
- 데이터 점검센터에서 엑셀 붙여넣기 데이터 문제만 필터 가능
- 전체 JSON 백업에 추적 ID를 선택적 metadata로 포함하고 집 복원 시 함께 복원
- 지원자 JSON 필드와 Supabase 테이블 구조는 변경하지 않음

## 호환성
- 기존 applicants / schools / employees / calendarEvents 구조 유지
- localStorage 기존 핵심 키 유지
- 백업 metadata가 없는 구형 JSON도 기존 방식으로 처리
